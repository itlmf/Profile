# API (Arabic/English-ready backend)

Base URL: `/`

## Health
- `GET /health`

## Trainees
- `POST /api/trainees`
- `POST /api/trainees/bulk`
- `GET /api/trainees/search?q=...`

Example single add payload:
```json
{
  "full_name": "اسم رباعي كامل هنا",
  "governorate_id": 1,
  "center_id": 1,
  "national_id": "30101010100123",
  "college_id": 1,
  "university_id": 1,
  "is_sonaa_volunteer": true,
  "sonaa_volunteer_date": "2024-07-01",
  "triful_join_date": "2024-08-15",
  "batch_id": 1
}
```

## Events + Attendance
- `POST /api/events`
- `POST /api/events/{id}/attendance`
- `GET /api/attendance/problems`

## Weekly Evaluations
- `POST /api/evaluations/weekly`

## Integrations
- `GET /api/integrations/external-sql?endpoint=trainees`
- `GET /api/integrations/kobo`

## Notes
- National ID validated as Egyptian 14-digit with valid date extraction.
- Trainee code generated by governorate code + serial + month/year.
- Cumulative evaluation percentage calculated from weekly 5-criteria score.
